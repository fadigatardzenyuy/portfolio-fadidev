# FadidevStudio - Abdul Fadiga Portfolio

A hyper-modern, bento-grid style portfolio for Abdul Fadiga, a Full-Stack Web Developer and UI/UX Engineer.

## Tech Stack

- **Framework**: Next.js 16 (Turbopack)
- **Styling**: Tailwind CSS v4
- **Animation**: CSS Keyframes + Framer Motion concepts
- **Theme**: Deep Dark Glassmorphism (`#030303`)

## Features

- **Bento Grid Layout**: Modular, responsive component grid.
- **Micro-Interactions**: Hover reveal effects and magnetic elements.
- **Performance**: Static Export ready, optimized images.
- **Resume System**: Integrated print-friendly resume page at `/resume`.

## Getting Started

```bash
npm install
npm run dev
```

## Deployment

Ready to deploy on Vercel:

1. Push to GitHub.
2. Import project in Vercel.
3. Framework Preset: Next.js.
4. Deploy.
